<INSTRUCTIONS>
## Project Rules
- Always put JavaScript in separate `.js` files under `src/js/`.
- Use Tailwind CSS only.
- Do not write custom CSS unless it is truly impossible with Tailwind. If needed, ask first. If so, put custom CSS in seperate `.css` files under `src/css/`.
- Never run build commands yourself, or change files in the docs folder. The user runs `npm run dev`, which auto-builds and copies to the docs folder automatically.
</INSTRUCTIONS>
